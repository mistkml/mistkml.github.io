mistk.orchestrator.template_env module

The base environment which has the expected env methods pre-defined.

Each game’s and/or video game engine’s orchestrator should be linked up to a game or environment. That game or environment’s class should inherit off of this BaseEnv class, which defines the core APIs we expect.
