mistk.watch.wait module

mistk.watch.wait.wait_for_state(service, stage, state)

Waits for a model or transform service wrapper to change their state to the desired state

Parameters:

  • service – The model, transform, evaluation, agent, or orchestrator service to get status for

  • stage – The current stage of the container

  • state – The desired state of the container
